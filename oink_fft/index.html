How To Use OINK FFT

1. Installation

To use this library, you first need to include it in your project.

	    	
	    		import * as OINK from 'oink_fft.js';

2. Initialization

Now, to make the FFT functions accessible, you need to call the WASM setup.

	    	
	    		const Module_OINK = import('fft_wasm.js');
	    	    initializeModuleOINK(Module_OINK.default());

3. Usage

Here's how you can use the Forward FFT of size 1024

	    	
	    		let realInput = new Float32Array([/* Fill with 1024 Float Values */]);
	    	    let spectrum  = fftReal1024(realInput);
