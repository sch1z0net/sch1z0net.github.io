How To Use OINK FFT

1. Installation

	    	
	    		//To use this library, you first need to include it in your project.
	    		import * as OINK from 'oink_fft.js';

2. Setup

	    	
	    		//Now, to make the FFT functions accessible, 
	    	    //you need to import the WASM file and call the WASM binding setup.
	    		const Module_OINK = import('fft_wasm.js');
	    	    initializeModuleOINK(Module_OINK.default());

3. Usage

	    	
	            //Here's how you can use the Forward FFT of size 1024
	            //It returns the spectrum, a Float32Array of length = 2048
	    		let realInput = new Float32Array([/* Fill with 1024 Float Values */]);
	    	    let spectrum  = fftReal1024(realInput);

Provided FFTs for different sizes:

	    	
	    		// 128 input -> 256 spectrum
	    		let spectrum  = fftReal128(realInput);
	    		// 256 input -> 512 spectrum
	    	    let spectrum  = fftReal256(realInput);
	    	    // 512 input -> 1024 spectrum
	    	    let spectrum  = fftReal512(realInput);
	    	    // 1024 input -> 2048 spectrum
	    	    let spectrum  = fftReal1024(realInput);
	    	    // 2048 input -> 4096 spectrum
	    	    let spectrum  = fftReal2048(realInput);
